# Истории активности

## 📌 Описание проекта

Этот скрипт позволяет автоматически создавать серию git-коммитов с искусственными датами в прошлом. Полезно, если нужно заполнить историю репозитория для демонстрации активности.

## ⚠️ Внимание

Скрипт создаёт коммиты с искусственными датами. Используйте только для тестовых или личных проектов.

## ⚙️ Технологии

- Python 3

## 🚀 Использование

1. Склонируйте репозиторий и перейдите в папку проекта:

   ```bash
   git clone <ваш-репозиторий>
   cd <папка>
   ```

2. Убедитесь, что у вас настроен git-репозиторий с удалённой веткой `main`.

3. Запустите скрипт:

   ```bash
   python main.py
   ```

По умолчанию в `main()` вызов:

```python
make_commits("00.00.0000", 0)
```

нужно заменить на свои параметры.

### Пример

```python
make_commits("01.01.2020", 10)
```

Этот вызов создаст 10 коммитов начиная с **01.01.2020** по **10.01.2020** включительно.
